# 🌐 Hanzala's Portfolio

This is my personal portfolio website — a single-page responsive site built with **HTML, CSS, and JavaScript**.  
It showcases my skills, projects, and provides a way to download my resume.

## 🚀 Features
- Responsive single-page design (mobile & desktop friendly)
- Smooth navigation to sections (About, Skills, Projects, Resume, Contact)
- Downloadable resume in PDF format
- Contact form (frontend only — can be connected to backend/email service)
- Clean and professional look

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox & Grid for responsiveness)
- JavaScript (for form and interactivity)
